# User Requirements

### User Requirement For Version 0.0.1

1. The system shall allow users to explore file system of a Linux operating system. 
2. The system shall remember the user's current location between commands, but shall reset to the user's home directory if the computer is rebooted.
3. Starting path of the system should begin with users home directory.
4. The system should be able to list all files and folders in a well formated, structured and human-readable way.
5. When the system list files, system shall list file name, file size (in appropriate units), last modified date, file path from root directory.
6. When the system list folders, system shall list folder name, cumulative size (including contents), created date of the folder.
